# Data Science Portfolio

This portfolio showcases my work across machine learning, analytics, and applied data engineering. Each project folder contains a Jupyter notebook with the full workflow, code, and findings.

## Projects

### 1. Customer Support LLM
**Large Language Model for Summarizing Customer Support Chats**

- Implements and fine-tunes transformer-based models (BART, T5) for text summarization on customer support data.
- Includes data cleaning, EDA, model training, ROUGE evaluation, and sample summary analysis.
- **Tech:** NLP, Hugging Face Transformers, EDA, model evaluation, prompt engineering

---

### 2. US Accident Analysis Project
**Exploratory Data Analysis and Predictive Modeling on US Traffic Accident Data**

- Full EDA and machine learning pipeline on US accident records.
- Covers data cleaning, visualization, feature engineering, and predictive modeling of accident severity/outcomes.
- **Tech:** pandas, matplotlib/seaborn, scikit-learn, classification, feature selection

---

### 3. User Engagement Project
**E-commerce User Behavior Analytics (Brandazon Platform Simulation)**

- Simulates user interaction data for a web/mobile e-commerce platform.
- Analyzes session behavior, churn prediction, campaign impact, and customer lifetime value.
- Uses sequence modeling and advanced feature engineering on synthetic datasets.
- **Tech:** Data synthesis, time series, ML modeling, churn analysis, campaign attribution

---

## Usage

- Notebooks are organized by project directory.
- Each notebook is standalone, with code, analysis, and results.
- Some data is synthetic to illustrate real-world analytics pipelines.
